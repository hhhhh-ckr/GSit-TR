## Local development

### Local project

Clone the repository:
```bash
git clone https://github.com/Gecolay/GSit.git
```

### Build

Run the `gradle build` command.

The final `GSit-x.x-x.jar` file will be in the [`build/libs`](./build/libs) folder.
